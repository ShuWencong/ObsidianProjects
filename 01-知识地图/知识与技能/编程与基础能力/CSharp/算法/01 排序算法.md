# 排序算法

你已经提供了几种排序算法的代码和简要说明，包括 **冒泡排序**、**选择排序**、**插入排序**、**希尔排序**、**快速排序** 等。我将逐一简要解释这些算法，并对你的代码中可能存在的问题进行优化或补充。

### 1. 冒泡排序（Bubble Sort）

- **时间复杂度**：最差情况和平均情况为 O(n²)，最优情况为 O(n)（优化后）。

- **稳定性**：稳定。

#### 普通冒泡排序（固定时间）

```csharp
public void BubbleSort(int[] res)
{
    for (int i = 0; i  res[j + 1])
            {
                int temp = res[j];
                res[j] = res[j + 1];
                res[j + 1] = temp;
            }
        }
    }
}
```

#### 优化后的冒泡排序（最快 O(n)，最慢 O(n²)）

   通过一个 `isSwap` 标志位，若一轮内没有发生交换，说明数组已经有序，可以提前退出。

```csharp
public void BubbleSort(int[] res)
{
    bool isSwap = false;
    for (int i = 0; i  res[j + 1])
            {
                int temp = res[j];
                res[j] = res[j + 1];
                res[j + 1] = temp;
                isSwap = true;  // 发生交换
            }
        }
        if (!isSwap) break;  // 若没有发生交换，退出
    }
}
```

### 2. 选择排序（Selection Sort）

- **时间复杂度**：O(n²)，无论是否优化。

- **稳定性**：不稳定。

#### 普通选择排序

```csharp
public void SelectionSort(int[] res)
{
    for (int i = 0; i  res[j])
            {
                minIndex = j;
            }
        }
        if (minIndex != i)
        {
            int temp = res[minIndex];
            res[minIndex] = res[i];
            res[i] = temp;
        }
    }
}
```

#### 优化后的选择排序

   可以同时查找最小和最大元素，减少遍历次数。

    b = temp;

### 3. 插入排序（Insertion Sort）

- **时间复杂度**：最差 O(n²)，最优 O(n)（已排序的情况）。

- **稳定性**：稳定。

#### 普通插入排序

```plain
public void InsertionSort(int[] res)
{
    for (int i = 1; i = 0 && res[j] > temp)
        {
            res[j + 1] = res[j];  // 将大的元素右移
            j--;
        }
        res[j + 1] = temp;  // 将 temp 插入到合适的位置
    }
}
```

- **时间复杂度**：最差 O(n²)，最优 O(n)（已排序的情况）。

- **稳定性**：稳定。

### 4. 希尔排序（Shell Sort）

- **时间复杂度**：根据步长序列而不同，通常在 O(nlogn) 到 O(n²) 之间。

- **稳定性**：不稳定。

```csharp
public void ShellSort(int[] nums)
{
    int distance = nums.Length / 2;
    while (distance > 0)
    {
        for (int i = distance; i = 0 && nums[j] > Base)
            {
                nums[j + distance] = nums[j];
                j -= distance;
            }
            nums[j + distance] = Base;
        }
        distance /= 2;
    }
}
```

### 8. 计数排序（Counting Sort）

- **时间复杂度**：O(n+k)，其中 n 是输入数组大小，k 是输入数组中的元素范围。

- **稳定性**：稳定。

### 9. 桶排序（Bucket Sort）

- **时间复杂度**：O(n+k)，其中 n 是输入数据，k 是桶的数量。

- **稳定性**：稳定。

### 10. 基数排序（Radix Sort）

- **时间复杂度**：O(nk)，其中 n 是输入数据，k 是最大数字的位数。

- **稳定性**：稳定。

这些是几种常见的排序算法及其 C# 实现，包含从基础的冒泡、选择、插入等 O(n²) 级别的排序算法到更高级的 O(nlogn) 级排序算法。
