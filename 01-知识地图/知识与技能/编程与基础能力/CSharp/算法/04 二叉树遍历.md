# 二叉树遍历

# 一、迭代法和递归法的区别

两种遍历方法的主要区别可以简化为以下几点：

### 1. 实现方式：

- **递归法**：通过函数自身调用，系统自动管理调用栈，代码简洁易读。

- **迭代法**：手动使用栈或队列来模拟递归，代码复杂度略高，但灵活可控。

### 2. 内存使用：

- **递归法**：使用系统调用栈，可能在树深度过大时导致栈溢出。

- **迭代法**：手动管理栈，避免栈溢出，更适合深度较大的树。

### 3. 可读性与复杂度：

- **递归法**：代码更简洁，逻辑直观。

- **迭代法**：代码稍复杂，但适合需要更精细控制的场景。

### 4. 性能：

- **递归法**：通常适合较浅的树，性能足够。

- **迭代法**：适合深度大的树，避免性能问题如栈溢出。

### 总结：

- **递归法**：简单、易读，适合一般树的遍历。

- **迭代法**：更灵活，适合处理深度大的树。

# 二、迭代法遍历

下面是三种遍历方式的代码，保持了相同的结构（包括 `null` 标记）。

### 1. 前序遍历（Preorder Traversal）：

```csharp
public IList PreorderTraversal(TreeNode root)
{
    var res = new List();
    var st = new Stack();
    if (root == null) return res;
    st.Push(root);

    while (st.Count != 0)
    {
        var node = st.Pop();
        if (node == null)
        {
            node = st.Pop();
            res.Add(node.val);
        }
        else
        {
            // 先处理根，所以先压入 null 标记和当前节点
            st.Push(node);
            st.Push(null);

            // 前序遍历顺序为 根-左-右，因此先压右子树，再压左子树
            if (node.right != null) st.Push(node.right);
            if (node.left != null) st.Push(node.left);
        }
    }
    return res;
}
```

### 2. 中序遍历（Inorder Traversal）：

```csharp
public IList InorderTraversal(TreeNode root)
{
    var res = new List();
    var st = new Stack();
    if (root == null) return res;
    st.Push(root);

    while (st.Count != 0)
    {
        var node = st.Pop();
        if (node == null)
        {
            node = st.Pop();
            res.Add(node.val);
        }
        else
        {
            // 中序遍历顺序为 左-根-右，因此先处理左子树，再处理根，最后是右子树
            if (node.right != null) st.Push(node.right);

            st.Push(node);
            st.Push(null);

            if (node.left != null) st.Push(node.left);
        }
    }
    return res;
}
```

### 3. 后序遍历（Postorder Traversal）：

```csharp
public IList PostorderTraversal(TreeNode root)
{
    var res = new List();
    var st = new Stack();
    if (root == null) return res;
    st.Push(root);

    while (st.Count != 0)
    {
        var node = st.Pop();

        if (node == null)
        {
            node = st.Pop(); // 再次 Pop 得到实际的节点
            res.Add(node.val); // 添加当前节点的值
        }
        else
        {
            st.Push(node);  // 将当前节点再次压入
            st.Push(null);  // 压入 null 标记

            // 按照根-右-左的顺序压栈
            if (node.right != null) st.Push(node.right);
            if (node.left != null) st.Push(node.left);
        }
    }

    return res;
}
```

### 说明：

- **前序遍历**：`根 -> 左 -> 右`，先访问根节点，然后左子树，最后右子树。

- **中序遍历**：`左 -> 根 -> 右`，先访问左子树，然后根节点，最后右子树。

- **后序遍历**：`左 -> 右 -> 根`，先访问左子树，然后右子树，最后根节点。

这三种遍历方式都保持了统一的结构，并且使用了 `null` 作为标记节点，确保遍历顺序一致。

# 三、递归法遍历

下面是使用递归法实现的三种遍历方式，分别是前序遍历、中序遍历和后序遍历。

### 1. 前序遍历（Preorder Traversal）：

前序遍历的顺序是 **根 -> 左 -> 右**。

```csharp
public IList PreorderTraversal(TreeNode root)
{
    var res = new List();
    PreorderHelper(root, res);
    return res;
}

private void PreorderHelper(TreeNode node, IList res)
{
    if (node == null) return;

    res.Add(node.val);           // 先访问根节点
    PreorderHelper(node.left, res);  // 递归遍历左子树
    PreorderHelper(node.right, res); // 递归遍历右子树
}
```

### 2. 中序遍历（Inorder Traversal）：

中序遍历的顺序是 **左 -> 根 -> 右**。

```csharp
public IList InorderTraversal(TreeNode root)
{
    var res = new List();
    InorderHelper(root, res);
    return res;
}

private void InorderHelper(TreeNode node, IList res)
{
    if (node == null) return;

    InorderHelper(node.left, res);  // 递归遍历左子树
    res.Add(node.val);              // 然后访问根节点
    InorderHelper(node.right, res); // 递归遍历右子树
}
```

### 3. 后序遍历（Postorder Traversal）：

后序遍历的顺序是 **左 -> 右 -> 根**。

```csharp
public IList PostorderTraversal(TreeNode root)
{
    var res = new List();
    PostorderHelper(root, res);
    return res;
}

private void PostorderHelper(TreeNode node, IList res)
{
    if (node == null) return;

    PostorderHelper(node.left, res);  // 递归遍历左子树
    PostorderHelper(node.right, res); // 递归遍历右子树
    res.Add(node.val);                // 最后访问根节点
}
```

### 说明：

- **前序遍历**：先访问根节点，再递归遍历左子树，最后递归遍历右子树。

- **中序遍历**：先递归遍历左子树，再访问根节点，最后递归遍历右子树。

- **后序遍历**：先递归遍历左子树，再递归遍历右子树，最后访问根节点。

这三种遍历方式都使用递归进行实现，代码结构清晰且易于理解。
