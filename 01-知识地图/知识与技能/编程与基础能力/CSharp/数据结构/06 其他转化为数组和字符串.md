# 其他转化为数组和字符串

为了便于记忆，使用相同或相似的 API 来将 **队列**、**栈** 和 **链表** 转换为数组和字符串。

### 1. 转化为数组

对于 **栈** (`Stack`)、**队列** (`Queue`) 和 **链表** (`LinkedList`)，我们可以统一使用 `ToArray()` 方法。这是因为 `Stack`、`Queue` 和 `LinkedList` 都支持将内容转换为数组。

#### 示例代码：

```csharp
// 栈
Stack stack = new Stack();
stack.Push(1);
stack.Push(2);
stack.Push(3);
int[] stackArray = stack.ToArray(); // 栈转为数组

// 队列
Queue queue = new Queue();
queue.Enqueue(1);
queue.Enqueue(2);
queue.Enqueue(3);
int[] queueArray = queue.ToArray(); // 队列转为数组

// 链表
LinkedList linkedList = new LinkedList();
linkedList.AddLast(1);
linkedList.AddLast(2);
linkedList.AddLast(3);
int[] linkedListArray = linkedList.ToArray(); // 链表转为数组
```

### 2. 转化为字符串

为了保持一致性，可以使用 `string.Concat()` 来将 **栈**、**队列** 和 **链表** 转换为字符串。如果需要为非字符类型（如整数）进行拼接，你可以先将它们转换为字符串，再使用 `string.Join()` 来拼接。

#### 示例代码：

```csharp
// 栈
Stack stack = new Stack();
stack.Push('a');
stack.Push('b');
stack.Push('c');
string stackString = string.Concat(stack); // 栈转为字符串

// 队列
Queue queue = new Queue();
queue.Enqueue('a');
queue.Enqueue('b');
queue.Enqueue('c');
string queueString = string.Concat(queue); // 队列转为字符串

// 链表
LinkedList linkedList = new LinkedList();
linkedList.AddLast('a');
linkedList.AddLast('b');
linkedList.AddLast('c');
string linkedListString = string.Concat(linkedList); // 链表转为字符串
```

### 对于非字符类型（如 int）转化为字符串

如果数据结构中的元素类型是非字符类型，比如 `int`，我们可以用 `string.Join("", ...)` 来拼接字符串，方便输出数字等。

```csharp
// 栈（int）
Stack stack = new Stack();
stack.Push(1);
stack.Push(2);
stack.Push(3);
string stackString = string.Join("", stack); // 栈中的数字转为字符串

// 队列（int）
Queue queue = new Queue();
queue.Enqueue(1);
queue.Enqueue(2);
queue.Enqueue(3);
string queueString = string.Join("", queue); // 队列中的数字转为字符串

// 链表（int）
LinkedList linkedList = new LinkedList();
linkedList.AddLast(1);
linkedList.AddLast(2);
linkedList.AddLast(3);
string linkedListString = string.Join("", linkedList); // 链表中的数字转为字符串
```

### 统一总结：

- **转化为数组**：

- 使用 `ToArray()` 方法。

- **转化为字符串**：

- 对于字符类型，使用 `string.Concat()`。

- 对于非字符类型，使用 `string.Join("", collection)`。
