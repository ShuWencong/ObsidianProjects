# 点乘与叉乘

# 1. 点乘与叉乘

## a. 点乘

在数学和计算机图形学中，点乘（也称为内积或标量积）通常用符号“·”来表示。在编程语言或软件工具中，点乘的表示可能会有所不同，例如：

### ⅰ. 数学表示

- 通常表示为𝑎⃗⋅𝑏⃗ ，其中 a,b 是向量。

### ⅱ. 在编程中的表示

- **C#（Unity）**: 使用 `Vector3.Dot(a, b)` 来计算两个向量 `a` 和 `b` 的点乘。

- **Python**（使用NumPy库）: 使用 `numpy.dot(a, b)` 来计算。

- **JavaScript**（例如在Three.js中）: 使用 `a.dot(b)` 来计算两个向量 `a` 和 `b` 的点乘。

### ⅲ. 点乘的计算公式

点乘的数学计算公式是：
a⋅b=∣a∣∣b∣cos(θ)
其中 ∣𝑎⃗∣ 和 ∣𝑏⃗∣是向量的模长，𝜃是这两个向量之间的角度。

如果将向量 𝑎⃗和 𝑏⃗分别表示为坐标形式 𝑎⃗=(𝑎𝑥,𝑎𝑦,𝑎𝑧) 和 𝑏⃗=(𝑏𝑥,𝑏𝑦,𝑏𝑧)，则点乘可以表示为： 𝑎⃗⋅𝑏⃗=𝑎𝑥×𝑏𝑥+𝑎𝑦×𝑏𝑦+𝑎𝑧×𝑏𝑧

![](https://cdn.nlark.com/yuque/0/2024/png/46628487/1721801211584-3ba6e79e-cfa2-4463-afc3-5a8b0b970843.png)

### ⅳ. 点乘的应用

点乘在计算和图形学中有多种应用，包括但不限于：

- **计算两个向量的角度**: 点乘的结果可以用来判断两个向量之间的角度关系，如是否正交或平行。

- **投影**: 计算一个向量在另一个向量上的投影。

- **光照计算**: 在计算光照模型（如Lambert光照模型）时，点乘用于确定表面的法线与光源方向之间的关系。

这种表示和应用确保了点乘在物理模拟、图形渲染、机器学习等领域的广泛使用。

## b. 叉乘

三维向量之间的叉乘（Cross Product）是一个基础且非常有用的数学操作，尤其在物理、工程、计算机图形学等领域。三维向量的叉乘产生一个新的向量，这个向量垂直于原始两个向量所在的平面。

### ⅰ. 定义与性质

![](https://cdn.nlark.com/yuque/0/2024/png/46628487/1721801289175-7c1999c6-ba36-4761-ab88-f7ab9b4fa1a2.png)

这个结果向量具有以下性质：

- **垂直**性：a×b 垂直于向量 𝑎⃗ 和 𝑏⃗。

- 方向：通过右手法则确定向量的方向。即如果你将右手的食指指向 𝑎⃗，中指指向 𝑏⃗，则拇指指向的方向即为 𝑎⃗×𝑏⃗的方向。

- 大小：𝑎⃗×𝑏⃗的大小等于 𝑎⃗和 𝑏⃗构成的平行四边形的面积，也即 ∣𝑎⃗×𝑏⃗∣=∣𝑎⃗∣∣𝑏⃗∣sin⁡(𝜃)，其中 𝜃 是 𝑎⃗和 𝑏⃗之间的夹角。

### ⅱ. 应用

- **计算法线向量**：在计算机图形学中，叉乘常用于计算多边形表面的法线向量，这对于光照和渲染至关重要。

- **旋转轴**：在三维空间中，当需要绕某个轴旋转对象时，可以使用两个向量的叉乘结果作为旋转轴。

- **物理和力学**：在力学中，用叉乘计算力的矩或角动量。

- **机器人学与工程**：用于计算各种机械臂和机构的关节和旋转部分的运动。

- **方向判断**：叉乘可以用来判断一个点相对于一个平面的方向，或者是两个向量相对方向的左右。

### ⅲ. 示例：计算叉乘的代码（C#）

```csharp
Vector3 CrossProduct(Vector3 a, Vector3 b)
{
    return new Vector3(
        a.y * b.z - a.z * b.y,
        a.z * b.x - a.x * b.z,
        a.x * b.y - a.y * b.x
    );
}
```

### ⅳ. 总结

三维向量的叉乘是一个多功能且强大的工具，它在许多科学和工程领域中都有广泛的应用。其结果的几何和物理意义使其在实际问题的解决中尤为重要。
