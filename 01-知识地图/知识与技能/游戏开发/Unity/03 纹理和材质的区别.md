# 纹理和材质的区别

在 Unity 和 3D 图形学中，**纹理（Texture）和材质（Material）是两个不同的概念**，而 **UV 信息是用于将纹理映射到 Mesh 表面的关键坐标系统**。以下是详细解释：

### 1. 纹理（Texture）与材质（Material）的区别

- **纹理（Texture）**

- 本质上是 **2D 图像**（如 PNG、JPG 等），存储颜色、法线、粗糙度等视觉数据。

- 作用：提供物体表面的细节（如木纹、金属划痕等），但本身不定义渲染行为。

- 示例：漫反射贴图（Albedo）、法线贴图（Normal Map）等。

- **材质（Material）**

- 是 **属性的集合**，包含纹理引用、光照参数（如金属度、光滑度）和着色器（Shader）。

- 作用：决定物体如何与光线交互（如反射、折射），纹理只是其输入的一部分。

- 关系：材质可以引用多个纹理（如颜色贴图+法线贴图），但纹理不能直接作为材质使用。

**关键区别**：

- 纹理是“数据”（图片），材质是“规则”（如何渲染这些数据）。

- 例如：一张木纹纹理（Texture）可以通过不同材质（Material）表现为“抛光木材”或“陈旧木板”。

### 2. UV 信息的作用

- **UV 坐标**

- 是 **Mesh 顶点上的 2D 坐标**（U 水平，V 垂直），范围通常为 `[0, 1]`，用于将纹理像素映射到模型表面。

- 作用：告诉引擎“纹理的哪个部分对应模型的哪个三角面”。

- **UV 映射示例**

- 若 UV 坐标为 `(0.5, 0.5)`，则取纹理正中心的颜色应用到对应顶点。

- 代码示例（Unity Shader 中的 UV 采样）：

```plain
fixed4 frag (v2f i) : SV_Target {
    return tex2D(_MainTex, i.uv); // 根据 UV 坐标采样纹理
}
```

- **UV 布局原则**

- 避免拉伸、接缝尽量隐蔽、充分利用 `[0,1]` 空间。

- 工具：Blender、Maya 等 3D 软件可编辑 UV 映射。

### 3. 三者协作流程

- **纹理** 提供图像数据（如颜色贴图）。

- **UV 坐标** 定义纹理如何包裹 Mesh（如模型展开为 2D 平面）。

- **材质** 通过着色器（Shader）将纹理按 UV 映射渲染到模型表面。

**示例**：

- 一个角色模型的皮肤纹理通过 UV 映射正确贴合到面部，材质则控制皮肤的光泽和透明度。

### 总结

- **纹理 ≠ 材质**：纹理是图片，材质是渲染规则。

- **UV 是桥梁**：将纹理精准定位到 Mesh 表面。

- **常见误区**：

- 误将纹理直接当作材质使用（需通过材质引用纹理）。

- UV 坐标范围超出 `[0,1]` 可能导致纹理重复或错乱。

如需进一步优化 UV 或材质效果，可参考 Unity 的 [UV 编辑工具](https://docs.unity3d.com/Manual/EditingUVs.html) 或 [Shader Graph](https://unity.com/shader-graph)。
