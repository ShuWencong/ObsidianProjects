# 柏林噪声

### Unity 中 Mathf.PerlinNoise 的使用整理

`Mathf.PerlinNoise` 是 Unity 提供的一个二维柏林噪声生成函数，用于生成平滑的伪随机值，广泛应用于地形生成、动画、程序化纹理等。

### 函数定义

```csharp
public static float PerlinNoise(float x, float y);
```

- `x`** 和 **`y`**：** 输入的二维坐标，用于生成噪声值。

- **返回值：** 返回一个范围在 `0`** 到 **`1` 的浮点值。

### 特性

- **确定性：**

- 对于相同的输入 `(x, y)`，始终返回相同的值。

- **平滑性：**

- 相邻坐标的输出值是连续的，变化平滑。

- **伪随机：**

- 虽然输出看起来随机，但实际上是由输入值决定的。

### 输入与输出特性

- **相同输入，相同输出：**

- `Mathf.PerlinNoise` 是确定性的，给定 `(x, y)` 的输入，结果始终相同。

- **输入范围：**

- 输入值可以是任意浮点数，Unity 不限制范围。

- 输入较大的值会导致密集的噪声变化，较小的值会让噪声变化平缓。

- **值范围：**

- 返回值在 `[0, 1]` 范围内。

- 如果需要负值，可以转换为：

```csharp
float noise = Mathf.PerlinNoise(x, y) * 2 - 1;
```
