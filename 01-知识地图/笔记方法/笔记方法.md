# 笔记方法

这个分支用来记录我怎样做笔记、怎样整理信息、怎样把零散内容变成可复用的知识。

## 基本原则

- 先记录，再整理
- 先链接，再分类
- 先形成真实使用习惯，再增加复杂结构
- 一篇笔记只解决一个主要问题

## 常用结构

一篇主题笔记可以先按这个顺序写：

1. 这是什么
2. 为什么重要
3. 我现在怎么理解
4. 可以怎么使用
5. 相关链接

## 整理流程

1. 新内容先放到最接近的主题或分支页
2. 如果内容变多，再拆成独立笔记
3. 拆出去的新笔记要链接回原来的分支页
4. 每周复盘时检查是否有孤立、重复或过期的笔记

## 和其他页面的关系

- 工具和软件使用经验，链接到 [[学习工具]] 或 [[Obsidian]]
- 学习方法和认知习惯，链接到 [[认知与学习]]
- 需要定期回看的整理任务，链接到 [[每周复盘]]

## 判断标准

如果一个内容回答的是“这条信息应该怎么记、怎么连、怎么复用”，就放在这里。
